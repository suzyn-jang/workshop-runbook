# Workshop Runbook - Review-Driven CTR

An interactive 90-minute workshop guide for designing review-driven click-through rate improvements.

## Overview

This repository contains a complete workshop runbook for facilitating a design thinking session focused on surfacing review signals in discovery experiences.

### Workshop Structure

- **Duration**: 90 minutes
- **Format**: Hybrid (in-person + remote)
- **Participants**: 4 groups, each with a unique persona
- **Output**: 4 concrete ideas grounded in real user needs

### Phases

1. **00 - OPEN THE ROOM** (0:00-0:05) - Orientation and group assignment
2. **01 - WOULD YOU GRAB?** (0:05-0:15) - Interactive icebreaker game
3. **02 - CONTEXT & INSPIRATION** (0:15-0:30) - Problem framing and persona distribution
4. **03 - GROUP IDEATION SESSION** (0:30-1:00) - Silent brainstorm, clustering, voting, pitch prep
5. **04 - GROUP IDEA SHARING** (1:00-1:20) - Group presentations
6. **05 - END** (1:20-1:30) - Next steps and wrap-up

### Personas

- 😩 **The Busy Worker** - Sick of the same food, no time to explore
- 🌟 **The Adventurous Foodie** - High standards, spends ages validating
- 🆕 **The New User** - No Grab history, only orders familiar places
- ✈️ **The Traveler** - Wants authentic local food, refuses chains

## Usage

Open `workshop-runbook.html` in any modern web browser. The runbook includes:

- ✅ Interactive checklist for facilitators
- ⏱️ Built-in timers for each activity
- 🎮 Embedded "Would You Grab?" game
- 📋 Persona cards and design challenges
- 📊 Visual timeline and phase navigation

## Features

- Responsive design for desktop and mobile
- No external dependencies - fully self-contained
- Timer management for time-boxed activities
- Progress tracking per phase
- Interactive card-swiping game for icebreaker

## How to Run

Simply open the HTML file in your browser:

```bash
open workshop-runbook.html
```

No server or installation required!
